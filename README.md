## Roll a dice

### Built with

- JS
- HTML
- CSS

GAME RULES:

- The game has 2 players and two dices, playing in rounds

- It takes an optional final score to set the winning score or takes default value of 100 as a winning score.

- In each turn, a player rolls a dice as many times as he wishes. Each result gets added to his ROUND score.

- BUT, if the player rolls a 1 on either of dice, all his ROUND score gets lost. After that, it's the next player's turn
- If 6 occurs on both dice at same time, All the scores, including global becomes zero.

- The player can choose to 'Hold', which means that his ROUND score gets added to his GLOBAL score. After that, it's the next player's turn.

- The first player to reach winning points on GLOBAL score wins the game.
